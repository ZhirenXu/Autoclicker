Before you start:
	Open CMD, type "pip install selenium". This command will install the key component of autoclicker.

	After that, find "firefoxdriver.exe" in the folder and run it. It should open another cmd window, output one line including "geckodriver" and a "Listening on" IP address. Close this CMD window. This driver is intented to install Firefox that can be opened and controlled by autoclicker.

How to run this script:
	To change privacy settings for OPPA records, run "AutoClicker for Privacy Setting.py". Then follow the instruction. When running, this script should open another firefox window, use our credential to log in DC collection and change records to private by click button by itself.
	
	Once all records in CSV files have been processed, the firefox window will close.
